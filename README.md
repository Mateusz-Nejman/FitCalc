# FitCalc
Extended JavaScript calories application, with offline support. 

## Features
- Responsive One Page application
- Nice charts
- BMR calculations
- Work offline
- Barcode scanner

## Used libraries
- Bootstrap
- jQuery with jQuery UI
- Chart.js
- Popper(for Bootstrap)
- QuaggaJs(Barcode scanner)
